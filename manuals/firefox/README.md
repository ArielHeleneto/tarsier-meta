# Firefox 测试

## 简介

Mozilla Firefox，通称 Firefox，中文也通称火狐，是一个自由及开源的网页浏览器，由 Mozilla 基金会及其子公司 Mozilla 公司开发。Firefox 支持 Windows、macOS 及 Linux。

## 测试目的

测试目标程序的安装和功能运行是否正常。

## 版本信息
